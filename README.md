# MERN Todo App

This is a simple MERN stack Todo application with authentication.

## Features
- User register and login using JWT
- Create, edit, and delete tasks
- Each user can access only their own tasks

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB

## Setup Instructions

### Backend
1. Go to backend folder  
2. Install dependencies  
   npm install  
3. Create a `.env` file and add:
   - MONGO_URL
   - JWT_KEY  
4. Start backend server  
   nodemon server.js

### Frontend
1. Go to frontend folder  
2. Install dependencies  
   npm install  
3. Create a `.env` file and add:
   - VITE_API_URL  
4. Start frontend  
   npm run dev
